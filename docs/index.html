뚜이 소개

뚜이는 스크래치 프로그래밍 언어를 확장한 것으로, 다양한 인공지능 및 데이터 수집 프로그램들을 만드는 데에 활용될 수 있습니다. 뚜이라는 가상의 친구와 대화하면서 실용적인 프로그램들을 만들어 보세요.

목차

  • 설치하기: 뚜이 프로그램 설치 방법을 안내합니다.

  • 블록 사용 설명서: 모든 블록 사용 방법을 설명합니다.

  • 처음 사용해 보기: 미션을 통해 간단한 예제 프로그램들을 만듭니다.

  • 예제 프로그램: 더 많은 예제 프로그램들을 소개합니다.

  • 프로그램 공유하기: HTML 파일과 실행 파일을 만드는 방법을 소개합니다.

  • 질문과 답변: 문의 방법을 안내합니다.
