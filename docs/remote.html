원격 접속 블록

  1. 아래의 파일을 클릭하여 내려받습니다.

  • Tooee_v6.0_RM.sb3

  1. 노란색 프로그램에서 파일 → 불러오기를 클릭하고, 내려받은 파일을 선택한 후 열기 버튼을 누릅니다.

Note

파일을 더블클릭하시면 잘 열리지 않을 때가 있습니다. 파일을 선택하신 후 열기 버튼을 클릭해 주세요.

이 블록들을 사용하려면 내 컴퓨터와 내 컴퓨터에 접속할 컴퓨터가 같은 와이파이를 사용해야 합니다.

블록이 잘 동작하지 않으면, 2.4G, 5G 등 여러 와이파이가 사용 가능한 경우 와이파이를 바꾸어 보세요.

원격 접속 허용하기 (선택사항)

  1. 원격 접속 허용하기 스프라이트를 클릭하고, 아래의 두 번째 블록을 클릭하여 원격 접속을 허용합니다.

Note

원격 접속 블록은 #원격이라는 값을 가지므로, #원격이라고 바로 타이핑해도 됩니다.

허용하지 않을 것 블록은 x라는 값을 가지므로, x라고 바로 타이핑해도 됩니다.

허용할 것 블록은 o라는 값을 가지므로, o라고 바로 타이핑해도 됩니다.

  1. 프로그램을 재시작합니다. 만약 아래와 같은 창이 뜬다면 엑세스 허용을 누릅니다.

내 컴퓨터의 주소 찾기 (선택사항)

  1. 내 컴퓨터의 주소 찾기 스프라이트를 클릭하고, 아래의 코드 블록을 각각 클릭합니다.

Note

내 컴퓨터 블록은 #주소라는 값을 가지므로, #주소라고 바로 타이핑해도 됩니다.

내 컴퓨터로 접속하기 (선택사항)

  1. 내 컴퓨터에 접속할 컴퓨터에서 내 컴퓨터로 접속하기 스프라이트를 클릭합니다. 아래의 코드 블록에서 192.168.0.37:9998 대신에 내 컴퓨터의 주소를 입력한 후 각 블록들을 실행합니다.

  1. 내 컴퓨터의 Data 폴더에 내 점수, 50000이 저장되었는지 확인합니다.

Note

접속할 컴퓨터 블록은 #주소라는 값을 가지므로, #주소라고 바로 타이핑해도 됩니다.

컴퓨터 보호를 위해 데이터 수집 블록들은 원격으로 실행할 수 없습니다.
