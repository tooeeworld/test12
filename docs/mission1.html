미션 1: 외계인 찾기

“오늘 여기서 외계인 친구들을 만나기로 했는데…

사실 난 외계인 친구들이 어떻게 생겼는지 몰라.

외계인 친구들을 찾을 수 있게 도와주지 않을래?”

Note

이 프로젝트는 웹캠이 필요합니다.

노트북 컴퓨터를 사용하시거나 웹캠이 설치된 데스크톱 컴퓨터를 사용해 주세요.

프로그램 소스 코드는 아래 링크를 통해 내려받으실 수 있습니다.

  • Tooee_v6.0_M1.sb3

초록색 깃발 버튼을 누르면, 웹캠이 켜집니다.

외계인 버튼을 누르면, 현재 화면에 보이는 사람이 외계인이라고 학습합니다.

지구인 버튼을 누르면, 현재 화면에 보이는 사람이 지구인이라고 학습합니다.

스페이스 키를 누르면, 화면에 보이는 사람이 지구인이라면 외계인 친구들이 인사하지 않습니다.

스페이스 키를 누르면, 화면에 보이는 사람이 외계인이라면 외계인 친구들이 인사합니다.

“고마워 친구! 난 이제 열기구를 타러 갈 건데 같이 가지 않을래?”
