뚜이 설치 방법

  1. 설치 파일을 더블 클릭하여 실행합니다.

  • Windows의 PC 보호라는 창이 뜬다면, 추가 정보 → 실행을 클릭합니다.

  • 게시자를 알 수 없는 이 앱이 디바이스를 변경할 수 있도록 허용하시겠어요?라는 창이 뜬다면, 예를 클릭합니다.

  • 바탕 화면에 바로가기 만들기가 선택되어 있습니다. 다음을 클릭합니다.

  1. 바탕화면에 아래와 같은 3개의 아이콘(Tooee, Data, HTML)이 있는지 확인합니다.

  1. Tooee 아이콘을 더블클릭하여 실행합니다. 작업표시줄에 아래와 같이 검은색 프로그램과 노란색 프로그램이 나타나는지 확인합니다.

Note

검은색 프로그램은 뚜이, 노란색 프로그램은 스크래치입니다. 두 프로그램 모두 작업표시줄에 있는지 확인해 주세요.

Data 아이콘을 더블클릭하면 Data 폴더가 나타납니다. Data 폴더는 인공지능 학습 시 활용됩니다.

HTML 아이콘은 HTML 파일을 만들 때 쓰입니다. HTML 파일 만들기를 참고해 주세요.

  1. 노란색 프로그램에서 CTRL 키와 + 키, CTRL 키와 - 키를 누르면 각각 화면이 커지고 작아집니다.

  1. 왼쪽 아래에 있는 확장기능 추가하기 버튼을 눌러보세요.

  1. 뚜이 확장기능을 선택해 주세요.

  1. 아래와 같이 3개의 블록이 추가되었다면, 사용할 준비가 완료되었습니다!
