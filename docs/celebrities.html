닮은꼴 연예인 찾기

Note

이 프로젝트는 웹캠이 필요합니다.

노트북 컴퓨터를 사용하시거나 웹캠이 설치된 데스크톱 컴퓨터를 사용해 주세요.

프로그램 소스 코드는 아래 링크를 통해 내려받으실 수 있습니다.

  • Tooee_v6.0_celeb.sb3

이 프로그램을 사용하시기 위해서는 4명의 연예인 사진들을 수집해야 합니다.

  • Data 폴더에 4개의 폴더(연예인1, 연예인2, 연예인3, 연예인4)를 만들고 사진들을 넣습니다.

  • 프로그래밍 화면에서 모양 탭을 누르고, 4개의 모양(연예인1, 연예인2, 연예인3, 연예인4)을 준비합니다.

초록색 깃발 버튼을 클릭하면, 연예인1~4 폴더에 저장된 사진들을 학습하고 비디오를 켭니다.

스페이스 키를 누르면, 아래 그림과 같이 가장 닮은 연예인의 사진(4개의 모양 중 하나)을 화면에 보여줍니다. (아래 그림에서는 저작권 문제로 인하여 연예인 사진을 일반 사진으로 대체하였습니다.)

Note

데이터 수집 블록을 이용하면 연예인 사진들을 쉽게 수집할 수 있습니다.

10,000개 이상의 데이터도 쉽게 학습시킬 수 있습니다.
